## Description

[Nest](https://github.com/nestjs/nest) 기반 잠수함 가상 훈련 시스템의 http 서버

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Build

```bash
# production build
$ npm run build
```
